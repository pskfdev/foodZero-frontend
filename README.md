# website foodZero (CRUD)

เป็นโปรเจคที่จัดทำขึ้นเพื่อเป็นผลงานในการฝึกเขียนเว็บไซต์ในส่วนหน้า โดยใช้ `React framework`

และมีการเขียนสำหรับติดต่อกับหลังบ้าน มีฟังก์ชั่นดังนี้

* Login / Logout
* CRUD User
* CRUD Product
* CRUD Category
* CRUD Reservation
* CRUD Blog

## language & library

* react
* react-redux
* css
  * react-bootstrap
  * sass
* axios
* react-read-more-read-less
* responsive
